此问题原因不清楚，一般来说就是微软的锅。

解决方法是：

1. 打开 Services.msc

2. 找到 Windows License Manager

3. 如果被禁用，右键点击并选择属性

4. 将启动类型设置为“自动” （重点）

5. 点击底部的“应用”然后“确定”保存

除此之外，还有问题就是包括防火墙、移动网络等服务无法启动，无法打开，界面直接卡死问题。

可以使用 windows 10 的 ISO 镜像文件进行对电脑修复。基本上都能解决问题。
